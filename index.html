功能与特点

稳定性

管理面板采用 Express 框架开发，绝大部分逻辑采用异步形式，确保主程序稳定运作。

安全性

在 Linux 下可使用 Docker API 实现虚拟化隔离，控制面板用户权限机制拥有严格检查。

开箱即用

无需数据库，无需 Apache/nginx ，可适用于 Linux/Windows，下载即用，一键安装！

界面美观

基于 Vue 的 SPA 网页，无需刷新，快速加载，并且拥有优雅的 UI 设计。
